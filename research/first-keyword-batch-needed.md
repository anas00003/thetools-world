## Manual Collection Notes

- Use incognito or a clean browser profile when checking Google results.
- Record country/region and date for Keyword Planner and Trends data.
- Keep competitor URLs exact, including whether the ranking page is a homepage, tool page, blog post, or category page.
- Separate true People Also Ask questions from related search suggestions.
- Prefer screenshots or exported CSVs where available so future analysis can be verified.
