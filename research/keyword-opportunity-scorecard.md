# Keyword Opportunity Scorecard

Source: manually provided SEMrush screenshot observations in `prompts/p7.txt`.

Scope: first scoring phase for TheTools.World image, compression, and bulk image cluster. This is not the final keyword plan for the whole site.

## Scoring Method

Each keyword is scored from 1 to 5 across the requested dimensions.

- Search intent clarity: whether the query clearly asks for a utility workflow.
- Traffic potential: visible volume adjusted for practical capture potential.
- Difficulty opportunity: lower KD and weaker likely SERPs score higher.
- Tool feasibility: whether TheTools.World can satisfy the query with a browser-based tool.
- AdSense value: expected monetization value based on CPC and commercial utility intent.
- Internal linking fit: whether the query fits the existing image tool cluster.

Total score is the sum of the six scoring columns. Decisions follow the prompt logic: build only for distinct high-intent utility problems, improve existing pages when they already match, merge close variants, delay later-category ideas, and avoid irrelevant or risky intent.
