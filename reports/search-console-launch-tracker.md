## Status Values

- `Pending manual check`: URL has not been inspected in Search Console yet.
- `Discovered - currently not indexed`: Google has found the page but has not indexed it.
- `Crawled - currently not indexed`: Google has crawled the page but has not indexed it.
- `Indexed`: Page is available in Google Search.
- `Blocked or error`: Search Console reports a crawl, canonical, robots, redirect, or server issue.

## Operating Notes

- Do not mark `URL inspected` as `Yes` until the live URL inspection has been run in Google Search Console.
- Do not mark `Indexing requested` as `Yes` until the request has been submitted for that exact URL.
- Keep notes factual: use Search Console wording where possible.
- Recheck all High priority URLs after 3 days and again after 7 days.
