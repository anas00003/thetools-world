## Avoid List

Do not target these in the current cluster:

| Keyword group | Examples | Reason |
| --- | --- | --- |
| Sharper Image product terms | sharper image air compression boots, sharper image compression boots, sharper image leg compression | Product shopping intent unrelated to image files. |
| Educational image searches | images of early human stone tools, hoe tool images, tools names and images | Informational image-search intent, not a utility workflow. |
| Downloader/scraper terms | bulk image downloader, download bulk image urls, bulk download images, tool to recursively save images from a webpage | Misaligned with the current utility brand and higher risk. |
| Service outsourcing | outsource bulk image editing services | Service intent, not a browser tool intent. |
| Broad seed term | image tool | Too mixed to target directly during this phase. |
